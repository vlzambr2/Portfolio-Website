Bunny Computer [WIP]

Goal: To deploy a fully-functional Raspberry Pi cluster inside of a custom-printed Retro Macintosh housing, capable of copying / loading music files in parallel from an external CD Drive. The Pi cluster would employ Chapel, an open-source parallel programming language, to enable parallelism between nodes. The finished project should be able to efficiently load and play music (from artists such as... the Beastie Boys ü§ò) out of the "Bunny Ear" speakers.

Last summer, I learned how to use Fusion360 to modify the project .STL files. In doing so, I resized the slot intended for a floppy disk reader to accomodate the CD Drive, designed new "Bunny Ear" attachments to hold the system speakers, and implemented a pin-based attachment system; I did this in order to maintain consistency with the original design, and to keep the parts modular.

The next steps are to compile Chapel on headless Raspian (a Linux distribution for Rasperry Pi), and begin programming the solution.

This project is an enhanced version of this Retro Macintosh single-board project by LowBudgetTech.

This project is a work in progress!
